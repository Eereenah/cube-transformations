# 3D transformations

Implementation of 3D transformations using homogeneous coordinates - rotation, translation and changing field of view. 

### Controls: 

* Rotation:
	* Around Y axis:
		* Clockwise: T
		* Counterclockwise: R
	* Around X axis:
		* Clockwise: E
		* Counterclockwise: W
	* Around Z axis:
		* Clockwise:        Y
		* Counterclockwise: U

* Camera translation:
	* X axis:
		* Right:S
		* Left: A
	* Y axis:
		* Up: F
		* Down: D		
	* Z axis:
		* Back: G
		* Front: H

* FOV:
	* Add angle: X
	* Subtract angle: Z
